## 📖 About

Electronics Hub is a comprehensive digital handbook for Electronics and Communication Engineering (ECE) students. It provides well-organized notes, formulas, definitions, and explanations for subjects taught throughout the four-year engineering curriculum.

The website is designed with a modern UI, responsive layout, dark/light theme, animated particle background, and an easy-to-use documentation interface.
